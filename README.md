# Bookstore API

This is a POC Bookstore API that allows a store owner to Add, Update, Find and Delete books in the bookstore

## Add

Endpoint: /book<br>
Method: POST

## Update

Endpoint: /book<br>
Method: PUT

## Find

Endpoint: /book<br>
Method: GET

## Delete

Endpoint: /book/{isbn}<br>
Method: DELETE
